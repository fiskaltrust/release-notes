# Middleware 1.3.79
[![Static Badge](https://img.shields.io/badge/milestone-v1.3.79-green?logo=github)](https://github.com/fiskaltrust/middleware/milestone/11?closed=1)

This release includes a new feature to automatically close open transactions on a TSE in the **German** market.

:::info

Version 1.3 of the Middleware is intended for the German and Italian markets only (with Belgium, Denmark, Greece, Portugal, and Spain currently in development).
Customers in Austria and France should continue to use version 1.2.
A preview of the unified 1.3 Middleware for Austria and France is already available.

:::


## ðŸ‡©ðŸ‡ª Feature: Automatically close open transactions on the TSE with the Daily-Closing ([fiskaltrust/middleware#554](https://github.com/fiskaltrust/middleware/pull/554))

We've added a queue configuration parameter `CloseOpenTSETransactionsOnDailyClosing` which is set to `false` per default.
When set to `true` the queue will close all open transactions on the TSE automatically with each daily closing.
